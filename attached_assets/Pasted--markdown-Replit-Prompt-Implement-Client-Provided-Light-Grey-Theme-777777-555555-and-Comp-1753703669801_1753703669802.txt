- **Inputs/Dropdowns:**

    - Text: `#fff` or dark text for strong contrast

## 4. Theme Examples

    ```
    // tailwind.config.js
    theme: {
      extend: {
        colors: {
          primarybg: '#777777',
          card: '#555555',
          border: '#6b6b6b',
          input: '#555555',
          button: {
            DEFAULT: '#555555',
            hover: '#fff',
            text: '#fff',
            hovertext: '#555555'
          },
          text: {
            DEFAULT: '#232323',
            light: '#fff'
          }
        }
      }
    }
    ```
- **Button class example:**
    ```
    bg-[#555555] text-white border-[#777777] hover:bg-[#fff] hover:text-[#555555] hover:border-[#555555]

    ```
